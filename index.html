Introduction to Programming


HTML

The web mainly consists of HTML docs (Hypertext Markup Languge). Here is the web Browsing Proccess:

User opens web browser > Browser send HTTPS request to server> Server retrieves content (HTML Doc) and forwards back to user
After the HTML doc is pushed to the user, the browser interprets the file and displays it on-screen.

- Text content
- Markup
- References
- Links

Attributes & Tags

- Attributes are used to assign certain properties to tags. The Anchoring attribute, for example, is used to create links. We can also assign
an Image attribute to a tag, by pulling its source.

- Tags are used to distinguish each type of element. Here are examples of some commonly used tags:

- Bold tags: Create bold text
- Emphasis tags: Italicise text
- Break tags: With this tag, we can specify when to move on to the next line.

Inline Vs. Block

Although they seem to perform the same function, they are actually quite different. As the name implies, Inline tage are kept in-line (localized).
Blocks are different, as they actually create a sort of "Invisible Box" around the designated HTML element.

WhiteSpacing

HTML interprets text on a single line by default. If we want to force multiple lines of text,
we have to use the break tag.
